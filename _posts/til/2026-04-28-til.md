# Today I Learned (2026-04-28)

## Barrel file과 CSS Grid

- `index` 파일을 여러 모듈을 다시 내보내는 배럴 파일(Barrel file)이라고 부른다는 것을 정리했다.
- CSS Grid에서 `minmax(min, max)`는 컬럼 크기를 최소값 이상, 최대값 이하로 조절할 때 사용한다.
- `fr`은 남은 공간을 비율로 나누는 단위이며, `1.2fr`은 기준 비율보다 조금 더 큰 몫을 차지한다.

## 코드 정리

- 여러 컴포넌트를 외부에서 가져다 써야 할 때는 배럴 파일로 export 경로를 정리하면 import 문이 단순해진다.
- 다만 배럴 파일이 너무 커지면 의존 관계가 흐려질 수 있어 도메인 단위로만 제한하는 편이 좋다.
- 복잡한 grid 템플릿 값은 상수나 의미 있는 class 조합으로 분리해 의도를 드러낼 수 있다.
