# Today I Learned (2026-03-11)

## MDX

- MDX는 Markdown + JSX를 함께 사용할 수 있는 포맷이다.

### 일반 Markdown

```md
# 제목

여기는 설명입니다.

- 리스트
- 리스트
```

### MDX

```jsx
# 제목

여기는 설명입니다.

<MyComponent />
<Chart data={data} />
```

### JSX

```jsx
export default function MDXContent() {
  return (
    <>
      <h1>제목</h1>
      <p>여기는 설명입니다.</p>
      <MyComponent />
    </>
  )
}
```
