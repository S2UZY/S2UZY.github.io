# Today I Learned (2026-05-29)

## X-헤더 (커스텀 헤더)

- Content-Type·Authorization처럼 정해진 표준 HTTP 헤더와 달리, `X-` 접두사를 붙인 헤더는 직접 정의하는 커스텀 헤더다. 클라이언트 종류·요청 ID 같은 부가 정보를 실어 보내고, 백엔드에서 `req.headers['x-...']`로 읽는다.

## nginx

- nginx는 리버스 프록시·정적 파일 서빙·로드밸런싱에 쓰이는 웹 서버다. 클라이언트 요청을 가장 앞에서 받아 뒤쪽 애플리케이션 서버로 넘겨주는 관문 역할을 한다.

## hono

- hono는 가볍고 빠른 웹 프레임워크로, Cloudflare Workers 같은 엣지 런타임에서도 잘 동작한다. 적은 코드로 API 서버를 만들 수 있다.
