# Today I Learned (2026-05-20)

## 토큰 만료와 Redis

- 토큰을 Redis에 저장해두면 만료 시점에 Redis에서도 함께 사라지도록 묶을 수 있다. 서버 저장소와 토큰 수명을 일치시켜, 만료된 토큰이 남아 오작동하는 걸 막는다.

## 동적 태그 컴포넌트

- 대문자로 시작하는 변수에 태그명 문자열을 담으면 JSX가 그걸 컴포넌트로 인식한다. 덕분에 조건에 따라 첫 줄은 `h1`, 나머지는 `p`처럼 시맨틱 태그를 동적으로 바꿀 수 있다. (변수명이 소문자면 일반 문자열로 취급된다.)

```jsx
const Tag = i === 0 ? 'h1' : 'p'
<Tag key={i}>{line}</Tag>
```

## git force vs force-with-lease

- `git push --force`는 원격을 무조건 덮어쓴다. `--force-with-lease`는 내가 마지막으로 받아온 이후 원격이 바뀌지 않았을 때만 밀어내, 그사이 올라온 동료의 커밋을 덮어쓰는 사고를 막아준다.
