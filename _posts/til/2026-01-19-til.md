# MCP 서버

AI가 외부 도구, 시스템, 데이터에 안전하게 연결되도록 중계하는 서버

기존에는 AI가 외부 시스템(DB, 파일, API 등)을 쓰려면:

- 서비스마다 제각각인 API
- 프롬프트에 임시로 규칙 주입
- 보안/권한 제어가 애매함

표준화 + 안전한 연결 방식이 필요했음

MCP 서버는 모델 소속이 아니고, 서비스 소유임
