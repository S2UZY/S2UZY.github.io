# Today I Learned (2026-06-18)

## Python with 문

- with는 객체의 `__enter__()`와 `__exit__()`을 자동으로 호출해주는 문법 설탕이다. 무언가를 열고 → 그 안에서 작업하고 → 빠져나올 때 정리(닫기)하는 흐름을, 중간에 예외가 나도 안전하게 처리해준다.

```python
with open("file.txt") as f:   # 들어갈 때 __enter__, 나올 때 __exit__ 자동 호출
    data = f.read()
```

## MCP

- MCP는 프롬프트가 아니라 "도구를 발견하고 호출하는" 프로토콜이다. 클로드가 "무슨 도구 있어?"라고 물으면 서버가 도구 목록을 주고, 실제 코드 실행은 그 MCP 서버 쪽에서 일어난다.
- 클로드에 MCP를 추가한다는 건 `mcpServers` 설정에 서버 실행 명령을 등록하는 일이다.

## Streamlit

- Streamlit은 파이썬 스크립트만으로 웹 UI를 만드는 프레임워크다. 핵심은 위젯을 건드릴 때마다 스크립트가 처음부터 끝까지 통째로 다시 실행된다는 점이라, 일반 변수는 매 실행마다 초기화된다.
- 그래서 값을 유지하려면 재실행돼도 살아남는 `st.session_state`를 쓰고, 무거운 로딩은 `@st.cache_data`·`@st.cache_resource`로 캐싱한다.

```python
if "count" not in st.session_state:
    st.session_state.count = 0   # 최초 1회만 초기화, 재실행돼도 보존
```
