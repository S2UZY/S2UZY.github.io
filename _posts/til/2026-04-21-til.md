# Today I Learned (2026-04-21)

## 청구서 디자인과 데이터 스텁

- 청구서 디자인 마무리 작업을 진행했다.
- 실제 데이터가 준비되기 전에는 데이터 스텁을 두고 화면 흐름과 레이아웃을 먼저 검증할 수 있다.
- 하네스 엔지니어링 관점에서 AI가 일할 환경 규칙과 구조를 설계하는 것이 중요하다는 내용도 함께 정리했다.

## 코드 정리

- 스텁 데이터는 화면 컴포넌트 안에 직접 두지 않고 fixtures로 분리하면 실제 API 데이터로 바꾸기 쉽다.
- 디자인 확정 전에는 데이터 형태를 먼저 맞춰두면 이후 컴포넌트 변경 범위를 줄일 수 있다.
