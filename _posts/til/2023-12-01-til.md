# object | Object | {} 차이

### primitive type

- number, boolean, string, symbol ...

### compound type

- function , object, class, array, regExp, tuple

## object

- compound type 만 가능
- primitive type 불가능
- 타입임

## Object

- compound type 가능
- primitive type 가능
- prototype 갖고 있음
- 그래서 new ~~ 가능

## {}

- compound type 가능
- primitive type 가능
- object literal 임
- - 참고 [] : array literal
- 프로토타입 없음
