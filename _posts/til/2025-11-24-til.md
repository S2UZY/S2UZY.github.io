### 개발하면서 느낀 것

1.  **type predicate** 어떤 값이 특정 타입인지 확인해주는 함수의 반환 타입 표기
    - predicate : 어떤 사실이 참인지 거짓인지 말해주는 것, 주어에 대해 설명해주는것

```ts
type Animal =
  | { type: "cat"; meow: () => void }
  | { type: "dog"; bark: () => void };

function isCat(animal: Animal): animal is { type: "cat"; meow: () => void } {
  return animal.type === "cat";
}

const pet: Animal = { type: "dog", bark: () => {} };

if (isCat(pet)) {
  pet.meow(); // ok!
} else {
  pet.bark(); // ok!
}
```
