# Today I Learned (2026-02-20)

## User-Agent

참고 링크
- [User-Agent 설명](https://www.maeil-mail.kr/question/289)

- User-Agent는 클라이언트가 서버 요청 시 헤더에 담아 보내는 사용자 환경 정보다.
- 브라우저 종류/버전, 운영체제, 디바이스, 렌더링 엔진 등의 정보를 포함한다.
- 프론트엔드에서는 브라우저 대응이나 디바이스 최적화에 참고할 수 있다.
- 다만 현대 웹 개발에서는 User-Agent 문자열에 과하게 의존하기보다, 기능 감지와 반응형 설계를 우선하는 것이 권장된다.
- JS에서는 `navigator.userAgent`, 서버에서는 HTTP 요청 헤더로 확인할 수 있다.

## 활용 경험

- 애널리틱스에서 사용자 환경 데이터 수집
- 특정 브라우저 렌더링 버그 대응
- 기능 미지원 환경에서 대체 UI 제공
