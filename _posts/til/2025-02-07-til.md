# 오늘 한 것

## GitHub upstream 설정

- fork 했던 저장소에 원본 저장소를 upstream으로 설정하는 방법

```bash
git remote add upstream https://github.com/original-repo.git
```

- remote 버전 확인

```bash
git remote -v
```

- remote upstream 삭제

```bash
git remote remove upstream
```
