# Date 시간 관련
- AWS 서버가 해외에 있다면 그 국가 시간으로 들어간다.
- 한국시간을 넣고 싶다면 아래와 같은 작업을 해줘야함

```js
export function getCurrentKTCTime() {
  return dayjs(new Date()).utc().tz('Asia/Seoul').toISOString();
}
```
