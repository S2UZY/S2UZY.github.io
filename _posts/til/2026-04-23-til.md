# Today I Learned (2026-04-23)

## 청구서 PDF 구현

- 청구서 PDF 구현을 진행했다.
- Claude Code 활용 방식과 정산 관련 API 흐름을 논의했다.
- PDF처럼 출력 결과가 중요한 기능은 화면 미리보기, 다운로드, 데이터 매핑, 페이지 나눔을 함께 고려해야 한다.

## 코드 정리

- API 응답 데이터를 PDF 템플릿에 바로 넘기지 않고, 화면에서 쓰기 좋은 형태로 매핑하는 레이어가 필요하다.
- 금액, 날짜, 사업자 정보처럼 반복 포맷팅되는 값은 유틸 함수로 분리하면 문서와 화면에서 함께 재사용할 수 있다.
- PDF 생성 로직은 브라우저 환경 의존성이 있으므로 클라이언트 전용 컴포넌트로 분리해 관리하는 편이 안전하다.
