# Today I Learned (2026-01-13)

# CPU 아키텍처(Instruction Set Architecture, ISA)

- CPU가 이해하는 명령어 규칙 + 레지스터 구조 + 메모리 접근 방식의 설계 표준을 말함
- x86 / ARM / PowerPC
  - 얘네가 종류 중 하나임

## 아키텍처가 정하는 것들

- 메모리 모델
- 레지스터 구조
- 엔디안 (Big/ Little Endian)

## 누가 정하냐?

- CPU 아키텍처 소유 회사가 정함
- 내부 구현은 상관없고 ISA (소프트웨어와 하드웨어 사이 계약서) 지키면됨
