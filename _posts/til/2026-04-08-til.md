# Today I Learned (2026-04-08)

## Admin 사용성 개선

- Admin 화면 사용성 개선을 이어서 진행했다.
- 콤보박스에 검색 기능을 추가해 옵션이 많아졌을 때도 원하는 값을 빠르게 찾을 수 있게 했다.
- 연동 신청 현황 테이블에는 승인/반려 타입을 추가해 상태를 더 분명하게 구분할 수 있도록 정리했다.

## 코드 정리

- 승인/반려처럼 반복해서 쓰이는 상태값은 문자열을 직접 쓰지 않고 타입과 상수로 분리하는 편이 안전하다.
- 테이블 컬럼 설정은 렌더링 코드 안에 길게 두기보다 별도 배열로 정리하면 컬럼 추가와 순서 변경이 쉬워진다.
- 검색 로직은 입력값 정규화와 필터링 조건을 분리해 테스트하기 좋은 형태로 다듬을 수 있다.
