## 2. git core.pare

- git log 볼 때, 페이저를 종료하려면 :q 를 입력하면 다시 돌아와야하는 불편함이 있었다.

- 아래 명령어 입력해서 설정해주면 바로 현재 명령어 창에서 보여줌

```jsx
git config --global core.pager cat
```
