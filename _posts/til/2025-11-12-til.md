## 개발하면서 느낀 것

1. Object.fromEntries -> 새로운 객체를 만드는 배열

```ts
const entries = [
  ["name", "수지"],
  ["role", "developer"],
];

const obj = Object.fromEntries(entries);

console.log(obj);
// { name: "수지", role: "developer" }
```
