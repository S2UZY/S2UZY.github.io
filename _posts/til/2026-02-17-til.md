# Today I Learned (2026-02-17)

## CSS Animation

- [MDN - CSS 애니메이션 사용하기](https://developer.mozilla.org/ko/docs/Web/CSS/Guides/Animations/Using)
- CSS 애니메이션은 `animation` 속성과 하위 속성들로 제어한다.
- 애니메이션 중간 상태는 `@keyframes` 규칙으로 정의한다.

```css
p {
  animation-duration: 3s;
  animation-name: slidein;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes slidein {
  from {
    margin-left: 100%;
    width: 300%;
  }

  to {
    margin-left: 0%;
    width: 100%;
  }
}
```
