# Today I Learned (2026-06-15)

## worktree로 클로드 CLI 여러 개 돌리기

- git worktree는 하나의 저장소를 여러 작업 디렉터리로 분리해주는 기능이다. 브랜치를 오갈 필요 없이 작업 공간이 따로 생겨, 클로드 CLI를 여러 개 동시에 띄워 작업할 수 있다.
- 구현 작업은 별도 스킬로 묶어, 최신 버전 pull → worktree 생성 → 작업 → PR → 변경 요약·리뷰 포인트 코멘트까지 한 흐름으로 돌리면 편하다.

## 클로드 스킬 모음

- 자주 하는 작업은 /skill-creator로 스킬을 만들어 자동화할 수 있다. 공개된 스킬 모음을 참고하면 직접 스킬을 만들 때 좋은 출발점이 된다.
