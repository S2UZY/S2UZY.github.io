3. npm install 과 npm ci 차이
   - npm ci 는 npm install에 비해 의존성의 버전을 엄격하게 유지함
   - npm install은 package.json에 적혀있는 버전 범위 내에서 다른 버전을 설치할 가능성이 있지만, npm ci는 오직 package-lock.json 에 정확하게 표기된 특정 버전을 따름
   - npm install은 package-lock.json을 변경할 가능성이 있지만, npm ci는 절대 변경하지 않음
   - npm ci는 node_modules을 삭제한 후 설치함
   - npm ci는 CI/CD 환경에서 빌드 과정의 일관성을 보장하기 위한 목적으로 많이 사용된다.
