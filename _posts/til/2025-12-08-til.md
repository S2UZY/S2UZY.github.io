## 특이사항

[Critical Security Vulnerability in React Server Components – React](https://react.dev/blog/2025/12/03/critical-security-vulnerability-in-react-server-components)

리액트 측에서 보안 취약점이 발생해서, 즉각적인 조치 필요하다고 공지함

**React (RSC) 취약점 대응 현황 정리**

1. 레거시 Admin 사이트

- 해당사항 없음
- 구버전 React 사용으로 **취약 버전(RSC 기반 React 19.x) 미사용**

2. CMS 사이트

- 조치 완료
- CMS는 **React Router 기반 CSR 앱**으로 RSC 미사용 → 구조적으로 취약점 영향 없음
- 보안 강화를 위해 React 버전을 안전 패치 버전(19.2.1)으로 업데이트 완료

3. 제니코드 AI Admin 사이트

- 조치 완료
- Nextjs, React 모두 보안 패치 버전(19.2.1) 으로 업데이트 완료
