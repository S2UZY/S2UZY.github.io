# Today I Learned (2026-06-17)

## GitHub App으로 봇 연동

- GitHub App은 사람 계정과 별개의 독립된 신원으로 레포에 접근하는 공식 확장 메커니즘이다. 필요한 권한만 콕 집어 주고, 특정 레포에만 설치하며, 이벤트를 구독(Webhook)해 반응한다. 개인이 퇴사하거나 토큰이 만료돼도 연동이 끊기지 않는다.
- 인증은 개인키로 JWT를 서명 → 그 JWT로 설치 토큰을 발급 → 설치 토큰으로 API를 호출하는 3단 구조다. 아래로 갈수록 권한이 좁고 수명이 짧아져, 키가 새어도 피해가 작다.

## @claude PR 코드리뷰 동작 원리

- 멘션을 감지하는 건 GitHub이다. 댓글 이벤트로 워크플로가 트리거되고, `if: contains('@claude')` 조건이 멘션 여부를 거른다.
- 실제 실행은 러너 VM 안의 Claude Code 엔진이 맡고, 내가 넣은 토큰으로 API를 호출하므로 비용은 내 계정에서 빠진다. 답글·코드 푸시를 GitHub에 쓸 때는 App 설치 토큰 권한으로 인증한다.
