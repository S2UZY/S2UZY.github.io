## rvest

- R 프로그래밍 언어를 위한 웹 스크래핑 패키지

**주요기능**

- HTML 페이지 다운로드 및 파싱
- CSS 선택자나 XPath 를 사용한 데이터 추출
- 테이블, 폼, 텍스트 등의 웹 콘텐츠 수집
- Python 의 Beautiful Soup 와 비슷한 기능을 R에서 구현

```tsx
library(rvest)

# 웹페이지 읽기
webpage <- read_html("https://example.com")

# CSS 선택자로 데이터 추출
title <- webpage %>%
  html_nodes("h1") %>%
  html_text()
```
