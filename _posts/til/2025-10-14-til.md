# Streaming SSR

한 번에 HTML을 보내는 것이 아니라, 준비된 부분부터 점진적으로 스트리밍해서 HTML을 보내는 것을 말한다.
React 18에서는 renderToPipeableStream API를 통해 구현할 수 있다고 함

```js
renderToPipeableStream(<App />, {
  onShellReady() {
    res.setHeader("Content-Type", "text/html");
    stream.pipe(res);
  },
});
```
